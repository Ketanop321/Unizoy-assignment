# Screenshots Guide

Add your final assignment screenshots in this folder.

Recommended file names:
- home.png
- jobs.png
- job-detail.png
- apply.png
- admin-login.png
- admin-dashboard.png
- admin-jobs.png
- admin-applications.png

Capture tips:
1. Use full browser width for consistency.
2. Capture one clean state per page.
3. Avoid showing secrets/tokens in URL or devtools.
